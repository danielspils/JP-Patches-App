# v0.8.1 — More reliable sequence captures

A tweak to Record-from-JX so dumps decode cleanly even when your JX-3P's
output runs a little quiet. If a sequence capture has ever failed on you
right after calibrating, this is for you.

## Install

Auto-update from v0.8.0 should bring this up automatically. Otherwise: DMG
below, drag to Applications. Signed + notarized.

**System requirement**: macOS 12+ on Apple Silicon (arm64).
