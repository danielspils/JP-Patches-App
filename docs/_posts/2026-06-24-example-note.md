This file is a template; it won't appear on the site (`published: false`).

To write a real note:

1. Copy this file to `_posts/YYYY-MM-DD-your-title.md` — the date in the
   filename is required, and it sets the post's date.
2. Edit the `title:` and `date:` in the front matter, and delete the
   `published: false` line.
3. Write your note in Markdown below the `---`.

The first paragraph becomes the excerpt shown on the Notes page, so lead
with a sentence that stands on its own.
