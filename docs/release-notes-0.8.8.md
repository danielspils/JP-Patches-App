# v0.8.8 — a small favor

**New:** JP Patches now sends one anonymous ping a day — just the app version and your country. No ID, no patches, nothing personal. It helps me see where JP Patches is popular. You'll be asked on first launch, and you can opt out any time in Settings.

**Also:** the gear icon is now "Settings" rather than "Audio settings" — it's where the ping toggle lives alongside the audio controls.

## Install

Auto-updates from v0.8.7. **System requirement:** macOS 12+ on Apple Silicon (arm64). Windows 10 (64-bit) or later.
