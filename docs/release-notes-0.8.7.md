# v0.8.7 — Windows tape capture + a snappier panel

**Windows:** JP now captures tape dumps from the JX-3P reliably on Windows at normal input levels with **no calibration needed**. App sounds (the sequencer editor and panel clicks) now route to your speakers correctly.

**Both platforms:**
- **Instant panel clicks** — button and switch sounds are now near-zero-latency.
- **Clearer tape dump timeline progress** — the Record timeline lights each section (Bank C, the divider, Bank D) exactly as that part of the dump comes in.

**Note:** The "what does my tape dump sound like?" feature is silent on the Windows version when sending from JX → JP. I'm working on a fix.

## Install

Auto-updates from v0.8.6. **System requirement:** macOS 12+ on Apple Silicon (arm64). PC beta: Windows 10 (64-bit) or later.
