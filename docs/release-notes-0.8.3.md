# v0.8.3 — Cleaning up dirty imports

Dropping or uploading the wrong file type (or a WAV corrupted by MP3 conversion
or stray audio) now shows a clear message, instead of freezing the window or
overwriting your banks.

## Install

Auto-update from v0.8.2 should bring this up automatically. Otherwise: DMG
below, drag to Applications. Signed + notarized.

**System requirement**: macOS 12+ on Apple Silicon (arm64).
